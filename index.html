Tutorial - Livre

Use a imaginação

  1. A partir de 1 jogador, recomendado até 4 jogadores
  2. Uso de regras fora do computador (a serem criadas)
  3. Recomendado para qualquer idade
  1. Arraste as instruções para o quadro do algoritmo na sequência que devem ser executadas.
  2. Para remover uma instrução do algoritmo dê um duplo clique.
  3. Terminado de criar a sequência clique no botão Executar.
  4. AGORA USE A IMAGINAÇÃO
  5. Crie as regras do jogo, como nos exemplos:
  6. Corrida de tabuleiro: Tenham um tabuleiro quadriculado em que o peão de cada jogador deve executar as instruções na tela até chegar em um determinado ponto;
  7. Desafio de movimentos: Os próprios jogadores ficarem de pé e se moverem conforme as instruções que aparecem na tela, com um jogador criando desafios para o outro.
  8. Etc...

Tutorial - Labirinto

Ache o caminho

  1. A partir de 1 jogador, recomendado até 3 jogadores
  2. Recomendado para pessoas a partir de 8 anos
  1. Arraste as instruções para o quadro do algoritmo na sequência que devem ser executadas.
  2. Para remover uma instrução do algoritmo dê um duplo clique.
  3. Terminado de criar a sequência clique no botão Executar.
  4. O seu objetivo é que a sequência de instruções leve o personagem até o quadrinho amarelo sem passar pelas paredes (quadrinhos escuros).
  5. Se o personagem errar o caminho retorne para a tela de edição e ajuste o algorítimo.
  6. Uma sugestão é jogar de forma cooperativa, com os jogares se alternando nos ajustes do algorítimo.

Tutorial - Movimento

Ande na linha

  1. A partir de 1 jogador, recomendado até 3 jogadores
  2. Recomendado para pessoas a partir de 10 anos
  1. Arraste as instruções para o quadro do algoritmo na sequência que devem ser executadas.
  2. Para remover uma instrução do algoritmo dê um duplo clique.
  3. Terminado de criar a sequência clique no botão Executar.
  4. O seu objetivo é que a sequência de instruções faça o personagem caminhar em cima da linha desenhada, formando figuras geométricas.
  5. Se o personagem errar o caminho retorne para a tela de edição e ajuste o algorítimo
  6. Uma sugestão é jogar de forma cooperativa, com os jogares se alternando nos ajustes do algorítimo

Tutorial - Ritmo

Mexa-se no Ritmo

  1. A partir de de 1 jogador, recomendado até 10 jogadores
  2. Uso de regras fora do computador (a serem criadas)
  3. Recomendado para qualquer idade
  1. Arraste as instruções para o quadro do algoritmo na sequência que devem ser executadas.
  2. Lembre-se que você pode usar o card "Ação" para criar instruções novas.
  3. Para remover uma instrução do algoritmo dê um duplo clique.
  4. Mova a barra azul para ajustar a velocidade
  5. Terminado de criar a sequência clique no botão Executar.
  6. AGORA USE A IMAGINAÇÃO
  7. Crie as regras do jogo, como nos exemplos:
  8. Desafio de movimentos: Os jogadores ficarem de pé e se moverem conforme as instruções que aparecem na tela, com um jogador criando desafios para o outro.
  9. Desafio crescente: Como no exemplo anterior, os jogadores ficarem de pé e se moverem conforme as instruções que aparecem na tela, porem com jogadores apenas adicionando uma instrução ao algorítimo a cada rodada, e não criando do zero.
  10. Dança em grupo (recomendado quando o número de jogadores é maior que 4): Um jogador será o mestre, o mestre cria o algorítimo, quando clica em executar os demais devem executar os movimentos;
  11. Etc...

Tutorial - Zumbi

Exploda as cabeças (de Zumbis)

  1. A partir de 1 jogador, recomendado 1 jogador
  2. Recomendado para pessoas a partir de 18 anos
  1. Arraste as instruções para o quadro do algoritmo na sequência que devem ser executadas.
  2. Para remover uma instrução do algoritmo dê um duplo clique.
  3. Terminado de criar a sequência clique no botão Executar.
  4. O seu objetivo é eliminar os zumbis sem tocá-los.
  5. O card "CORINGA" dispara um tiro com alcance de três unidades (quadrinhos).
